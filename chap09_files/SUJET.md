## Exercice 1 — File circulaire (5 pts)
`CircularQueue` : `enqueue`, `dequeue`, `front`, `is_empty`, `is_full`, `__len__`.

## Exercice 2 — File de priorité min-heap (5 pts)
`MinPriorityQueue` : `insert(val, priority)`, `extract_min()`, `peek_min()`.

## Exercice 3 — BFS + plus court chemin (5 pts)
- `bfs(adj, src)` → ordre de visite — O(V+E)
- `shortest_path(adj, src, dst)` → nombre d'arêtes minimum

## Exercice 4 — Parcours niveau par niveau (5 pts)
